14. Code Example

This JavaScript function creates a valid NFTx transaction packet to transmit encrypted payloads between programmable profiles.

const createNFTxPacket = ({
  sender,
  receiver,
  profileType,
  assetID,
  logicHeader,
  gasCredit,
  permissions,
  payload
}) => {
  return {
    protocol: 1,
    netcode: 42,
    sender,
    receiver,
    profileType,
    assetID,
    logicHeader,
    signature: sign(sender, payload),
    gasCredit,
    permissions,
    sync: Date.now(),
    payload
  };
};

15. Technical Deep Dive

üì¶ NFTx Packet Structure

packet-nftx:
  protocol_version: uint8      # 0-7
  network_code: uint8          # 8-15
  sender_wallet_id: uint256    # 16-47
  receiver_wallet_id: uint256  # 48-79
  profile_type: uint16         # 80-95
  asset_id: uint32             # 96-127
  smart_logic_header: bytes    # 128-159
  signature: bytes             # 160-191
  gas_credit: uint32           # 192-223
  permission_flags: uint16     # 224-239
  sync_pulse: uint16           # 240-255
  payload: bytes               # 256-511+

üîí Security Flow

  • Validate permission flags
  • Verify signature using profile public key
  • Decrypt payload (if private)
  • Execute smart logic header
  • Store logs on chain
  • Reward sender if applicable
